# About StaticPress

StaticPress is a local-first static site generator that lets you create websites and blogs with a WordPress-like experience, then deploy to Cloudflare Pages.

## Features

- **Local Admin Panel** - Create posts and pages in your browser
- **Markdown Editor** - Write content with easy formatting
- **Theme Customization** - Visual and code-based theming
- **One-Click Deploy** - Deploy to Cloudflare Pages withwrangler
- **Open Source** - Free to use and modify

## Tech Stack

- **Astro** - Static site generator
- **React** - Interactive admin UI
- **Tailwind CSS** - Styling
- **Bun** - Runtime
- **Cloudflare Pages** - Deployment

Get started by running `bun run dev` and visiting http://localhost:4321/admin
